=head1 Group::Git::Cmd::Stats

Adds the stats command to L<Group::Git> which allows you to collect statistics
accross many repositories.

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install
